# 📋 Task Board

Aplicação web de gerenciamento de tarefas (Task Manager) com interface estilo Trello, desenvolvida com HTML, CSS e JavaScript puro.

---

## 🚀 Funcionalidades

- Criar novas tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Filtrar tarefas (todas, pendentes e concluídas)
- Salvamento automático no navegador (localStorage)
- Interface estilo cards (Trello)

---

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

---

## 💡 Objetivo do projeto

Este projeto foi desenvolvido para praticar:
- Manipulação do DOM
- Lógica de programação
- Eventos em JavaScript
- Estrutura de aplicações web
- Persistência de dados no navegador

---

## 🎨 Interface

O projeto possui um design simples e moderno, com foco em usabilidade e visual estilo “kanban board”.
